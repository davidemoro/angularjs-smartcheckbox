angularjs smartcheckbox

Usage

Smart checkbox list directive for: 

 <smart-checkbox model="smartlist.mylist" requiredfn="smartlist.required" />
Parameters:
  • model, array of objects with if and label attributes
  • requiredfn (optional), required function with the current elem and a selected list. You can choose the required strategy of the checkbox list
where smartlist.mylist is: 
{{smartlist.mylist | json}}
with smartlist.required: 
    $scope.smartlist.required = function (elem, selected) {
        // elem is the current elem of the list
        // selected is an array of selected elems
        return angular.isArray(selected) ? selected.length === 0 : true;
      };
